# Action Careers - Modern Recruitment Website

A professional, modern recruitment website for Action Careers targeting young professionals. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Clean, professional UI with deep-navy headers, green buttons, and orange popups
- **Responsive Layout**: Fully responsive design works on all devices
- **Animations**: Smooth animations using Framer Motion to engage users
- **Job Search**: Comprehensive job search functionality
- **Job Categories**: Browse jobs by industry categories
- **Testimonials**: Carousel slider showcasing client testimonials
- **Interactive Elements**: Call-to-action sections and interactive UI components

## Tech Stack

- **Next.js**: React framework for production
- **TypeScript**: For type safety and better developer experience
- **Tailwind CSS**: For utility-first styling
- **Framer Motion**: For animations and transitions
- **React Icons**: For comprehensive icon library
- **Swiper**: For carousel/slider functionality

## Getting Started

### Prerequisites

- Node.js 14.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/action-careers.git
   cd action-careers
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Deployment

The site is configured to be easily deployed to Vercel or any other hosting platform that supports Next.js.

```bash
npm run build
# or
yarn build
```
